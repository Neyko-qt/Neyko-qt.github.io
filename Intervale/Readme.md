1. Установить node.js https://nodejs.org/en/
2. Для сборки я использовал webpack. Необходимо зайти в директорию с проектом и через консоль выполнить npm install webpack -D,  а также npm install webpack-cli -D. 
3. Далее через консоль использовать команду npm run start. Она соберет файл main.js. При изменении файлов(например в config.js смена цвета кнопки) вебпак автоматически будет пересобирать файл main.js.
4. Адаптивность я реализовал через медиавыражения CSS, при ширине окна менее 780 px все элементы перестраиваются как было в ТЗ. При этом инпуты с email и суммами переводов я сделал резиновыми (чтобы они тянулись при измении ширины контейнера).
